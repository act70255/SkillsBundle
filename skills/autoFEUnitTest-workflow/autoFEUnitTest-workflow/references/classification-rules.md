# Classification Rules

## 技術棧分類
- `React`
- `Vue`
- `HTML + JS`
- `HTML + JS + jQuery`

## 測試目標分類
- 純函式
- 工具模組
- DOM 操作
- UI 元件
- 事件互動
- 表單驗證
- 非同步邏輯

## 執行環境分類
- `node-only`
- `jsdom`
- `happy-dom`

## 外部依賴分類
- 可完全 mock
- 需部分 mock
- 需依契約生成
- 依賴瀏覽器能力模擬

## 風險分類
- 核心商業邏輯
- 高變動模組
- 歷史缺陷區域
- legacy 難測區塊

## 依據要求
每一個分類結果都必須記錄：
- 來源檔案或設定
- 推導理由
- 不確定性或風險
